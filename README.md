## 📚Study Flow

- 그 주에 해당하는 Chapter 를 ISSUE 로 생성한다.

  - ISSUE 는 장 단위로 생성한다.
  - ISSUE 에서는 해당 챕터에 대한 Q&A 를 진행한다.
  - WEEK Folder 의 README Q&A 링크로 달아 놓는다.

- Week Label 을 붙인다.

- 해당 Chapter를 맡은 담당자에게 Chapter Issue 를 할당한다.

- 폴더는 Week 단위로 생성한다.
  - 각 그룹원의 폴더가 하나씩 생성됨
  - 책을 읽다가 요약 또는 흥미로운 부분에 대해서 정리해오고, 발표하는 형태로 진행

<br>
